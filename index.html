📋 How to update after each matchday Show / Hide
  1. Open data.json — update "last_updated" and "matchday"
  2. Add two entries to "teams" per completed match — include sot (shots on target), corners, spg (set piece goals), plus xgf/xga/poss/margin
  3. Remove the team from "unplayed" once it has match data
  4. Set "est": false once stats are Opta-confirmed
  5. Commit and push to GitHub — site updates automatically in ~60s
